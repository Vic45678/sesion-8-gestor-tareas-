# Gestor de Tareas

## Descripción
Este proyecto es una aplicación de gestión de tareas desarrollada en Python. Permite a los usuarios agregar, listar y completar tareas de manera eficiente a través de una interfaz de consola. Fue creado como parte del reto integrador de la Unidad 1 para consolidar buenas prácticas de desarrollo, control de versiones y calidad de código.

## Requisitos
Para ejecutar este proyecto, asegúrate de tener instalado:
* **Python 3.x** (Recomendado 3.14.3 o superior).
* **Git** instalado en tu sistema.
* Un editor de código (como VS Code).
